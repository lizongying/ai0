# ai0

本工具包裝了DeepSeek等大模型，可以在群組裡指揮他們，讓他們為你工作；可以一鍵導出到word、excel、txt等，並保留美化後的樣式。

- 不收集任何用戶信息，完全本地運行
- 支持多個大模型單獨或同時發起請求
- 支持一鍵複製到word、excel、txt
- 支持一鍵下載MarkDown格式到本地
- 支持大模型返回數據自動保存到本地
- 支持群組對話方式樣式，表現更有趣
